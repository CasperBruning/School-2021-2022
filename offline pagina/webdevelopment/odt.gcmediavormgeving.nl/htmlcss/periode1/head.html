HTML & CSS - Head

De head

Tussen <head> & </head> kan je een aantal "eigenschappen" geven. Hieronder zie je een aantal eigenschappen die je in de HEAD-sectie zou kunnen gebruiken. Alleen de 1e (<title>) is verplicht.

tags: Doel:
<title>Titel van pagina</title> Hiermee geef je je site een “titel”. Deze staat boven in de browser als je webpagina word geladen.
<style type=”text/css”></style> Tussen deze tags kan je CSS code zetten waarmee je de webpagina kan opmaken.
<link rel="stylesheet" type="text/css" href="mijnstyle.css" > Hiermee koppel je een “externe” CSS bestand (stylesheet) aan je webpagina.
<meta name="description" content="Beschrijving" > Met deze tag geef je de webpagina een beschrijving. Vooral belangrijk voor zoekmachines.
<meta name="author" content="Jouw naam" > Met deze tag kan je vermelden dat jij de auteur bent. Vooral belangrijk voor zoekmachines.
<meta name="keywords" content="Zoekwoorden" > Met deze tag geef je een aantal zoekwoorden op. Zoekmachines registreren dit en zodra er op deze woorden gezocht wordt, komt jou webpagina eerder naar voren.

Alles wat in de HEAD-sectie staat is vooral belangrijk voor zoekmachines. We gaan nu nog niet heel veel aandacht aan de head besteden. Het is belangrijk om te weten waar het voor dient.

Als je goed kijkt zie je dat de tag <meta> meerdere malen voorkomt. Je ziet dat er verschillende eigenschappen aan de tag zitten. Dit gaan we nader bekijken.

Opdracht 1: meta tag

Open in je editor het bestand Opdracht1-2.html

  • Sla het bestand op als Opdracht2-1.html
  • Ga met je cursor achter de sluittag van title staan (</title>) en druk op enter.
  • Type nu <meta en daarna een spatie.
  • Als je met Dreamweaver werkt merk je dat er een uitvouwmenu verschijnt.
  • Type nu name en druk op de tab-toets.
  • Je ziet dat er automatisch =”” Ook staat je cursor nu precies tussen de twee aanhalingstekens in. Dat is ook de bedoeling. Je ziet ook een nieuw uitklapmenu verschijnen.
  • Ga met de pijltjes toets naar description en druk op enter. Mocht je uitklapmenu verdwenen zijn, kun je deze altijd oproepen door op ctrl+spatiebalk te drukken.
  • Type vervolgens weer een spatie en selecteer nu content. Druk weer op enter.
  • Type nu weer tussen de twee aanhalingstekens een beschrijving van deze opdracht.
  • Als je daarmee klaar bent ga je achter het laatste aanhalingsteken staan en type je een >. Hiermee sluit je de meta tag af.

Sla het bestand op als Opdracht2-1.html.

Je ziet dat de meta tag niet op de originele manier wordt afgesloten. Je zou verwachten aan het einde van de regel </meta>. Dit is alleen het geval wanneer de tekst tussen de begin- en sluittag beïnvloed wordt door de tag. In dit geval staat alle info al in de eigenschappen dus wordt de tag niet afgesloten.

Let op:
Ook de <br> (enter op je scherm) en <hr> (horizontale lijn) tags worden niet afgesloten!

Bijna elk html element heeft verschillende eigenschappen. Om te kijken welke eigenschappen er mogelijk zijn kun je altijd met je cursor voor de > van de begin-tag gaan staan en een spatie typen. In Dreamweaver verschijnt dan het uitvouwmenu, ook wel de codehint genoemd.

Opdracht 2: meta tag uitbreiden

Open in je editor Opdracht 2-1.html

  • Voer een nieuwe meta tag in met bij name Author en bij content jouw naam.
  • Voer nog een nieuwe meta tag in met keywords en een aantal zoekwoorden die over deze opdracht gaan. De zoekwoorden moeten met een komma gescheiden worden.
  • Probeer zoveel mogelijk gebruik te maken van de codehint. Lukt dit niet, type het dan zelf. Kijk daarbij goed naar waar je wat neer moet zetten.
